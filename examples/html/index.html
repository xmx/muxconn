MUXCONN · 连接多路复用器

一个 Go 库，可将多个子流多路复用到单个连接上，底层支持 yamux 或 smux。

REST API

GET /api/clients GET /api/limit?id={id}&limit={rate} GET /api/kill?id={id}&sid={sid} GET /api/tunnel GET /api/direct/{path}?id={id}

限流测试

客户端上线后，使用 curl 测试限流效果。在页面上调整限流值，观察下载速度的变化。

Linux / macOS

curl -L "http://127.0.0.1:9999/api/direct/api/download?id={CLIENT_ID}" -o /dev/null

Windows

curl.exe -L "http://127.0.0.1:9999/api/direct/api/download?id={CLIENT_ID}" -o NUL
